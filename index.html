HTML Tables

The <table> element represents tabular data (information presented in a two-dimensional table comprised of rows and columns of cells containing data).

Tables should only be used for tabular data, not layout.

Basic Structure

A table is defined with the <table> tag, and contains rows (<tr>), which in turn contain cells (<td> for data, <th> for headers). A table cell can contain all sorts of HTML elements: text, images, lists, links, other tables, etc.


The <thead> element is used in conjunction with the <tbody> and <tfoot> elements to specify each part of a table (header, body, footer). These elements don't change the visual order but help with semantics.


A <caption> element with a clear and concise value describes the purpose of the table. This can help people who navigate with the aid of assistive technology such as a screen reader.

Table Tags

Common HTML Table Tags
Tag Name Description
<table> Table The element that wraps the whole table.
<thead> Table Head Groups the header rows.
<tbody> Table Body Groups the body rows.
<tfoot> Table Foot Groups the footer rows.
<th> Table Header Cell Defines a cell as the header of a group of table cells and may be used as a child of <tr>.
<tr> Table Row The table row container (contains <th> or <td> cells).
<td> Table Data The table data cell.
<caption> Table Caption Defines the caption (or title) of a table, providing an accessible description.
<colgroup> Table Column Group Defines a group of columns within a table.
<col> Table Column Defines one or more columns in a column group.

Table Attributes

Common HTML Table Attributes
Attribute Name Description
colspan Column Span Defines how many columns a <th> or <td> element should span.
rowspan Row Span Defines how many rows a <th> or <td> element should span.
